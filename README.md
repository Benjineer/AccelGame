AcceleratorPlus 2

A simple one-class game written in Java.

![screen shot](screenshot_accelP2.PNG?raw=true "AcceleratorPlus 2")

Instructions:
Just download and stick it into an eclipse project... and run.

Game-Play:
  >Up, Down, Left and Right - Arrow Keys
  >Space                    - Shoot
  >W, D                     - Accelerate
